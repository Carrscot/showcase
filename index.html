Showcase - Scott Carroll
This is a collection of projects I have done throughout the course of my learning experience with The Odin Project. I wanted to keep a record of my progress to see everything I have learned aalong the way and how far I have come. The projects are currently listed from most recent to my first project which is the final entry. Please feel free to click and explore as much as you would like.
Calculator
A very basic calculator using basic HTML, CSS and javascript. This was my first proper project using DOM manipulation in conjunction with programming javascript logic.
Etch-A-Sketch
Etch-A-Sketch project where the main challenge was using DOM manipulation methods to create many objects as efficiently as possible and using event listeners to dynamically alter CSS.
Rock, Paper, Scissors
This rock paper scissors project was my first introduction to javascript, its uses as well as creating an interactive UI.
Landing Page
This was my first full CSS project where I was tasked with recreating a given landing page for a website. The main challenge was figuring out the many uses and tricks of flexboxes and how to manipulate them in order to create a precise and neat looking page.
Recipes
My first project for The Odin project was to create a basic webpage with different recipes. The main objectives were to learn how to create elements and manipulate text and images in CSS.
